python生成个性二维码
2020-04-21

1.引言

这张二维码是不是很炫酷呢，没错，我们常见的二维码都是静态的，一动不动，感觉缺少点创意，今天就让我来给大家演示一下动态二维码是怎么做出来的。

2.所需要的工具
①Python + MyQR库
②二维码(例如QQ，微信二维码等)
③所需网站：https://cli.im/  (主要将二维码转换为网址)
④自己喜欢的图片，包括GIF格式的(做动图需要)

3.二维码转换为网址步骤

4.代码

import os

from MyQR import myqr

myqr.run(

words='https://qm.qq.com/cgi-bin/qm/qr?k=qFHNyo5r5pOdATTJteyLl1Rc6dgo1ogo', ###扫描二维码后要跳转的链接

picture='/home/apollo/python小实验/cailiao01.gif',

version=5,##容错率

level='H',###纠错水平，范围是L，M，Q，H，从左到右依次升高

colorized=True,###False 为黑白，True为彩色

contrast=1.0,###调节图片,1.0表示原始图片

brightness=1.0,###调节图片亮度

save_name = '/home/apollo/python小实验/QQ.gif' #图片后缀一定要和上面一致

6.补充

静态个性二维码的制作和动态艺术二维码一样，只是要保证上传的图片格式是jpg或者png格式即可，输出保存时后缀也要跟上传的图片格式一致。
